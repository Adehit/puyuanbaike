# 学生会

莆田一中学生会是福建省莆田第一中学涉及面广的学生组织，也是莆田一中历史悠久的学生组织。莆田一中学生会始终以为同学服务为宗旨，维护学生利益，丰富学生的课余生活。作为学校内活力的学生组织，学生会**举办了校内大多数学生活动**，例如**达人秀**、**各项体育赛事**、**各类才艺比赛**等。
